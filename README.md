## Общая информация

**Happ** — кроссплатформенное приложение на основе **Xray**, предназначенное для работы с сетевыми конфигурациями, прокси-профилями и параметрами подключений. Клиент объединяет инструменты для создания, добавления, импорта, настройки, редактирования и хранения профилей, позволяя управлять несколькими вариантами соединений через единый интерфейс.

Благодаря поддержке современных протоколов, гибким возможностям маршрутизации и расширенным инструментам конфигурирования **Happ** подходит как для быстрого использования готовых настроек, так и для случаев, когда требуется более точная настройка параметров соединения и контроля сетевого трафика.

Пользователь может создавать отдельные конфигурационные наборы под разные задачи, сохранять их локально и быстро переключаться между необходимыми профилями в зависимости от текущих условий.

## Основной функционал

* Создание индивидуальных профилей с заданными параметрами подключения.

* Добавление конфигураций через импорт готовых данных или ручное заполнение настроек.

* Изменение и настройка существующих вариантов соединений.

* Управление правилами маршрутизации и обработкой сетевого трафика.

* Поддержка популярных протоколов:

  * **VLESS Reality**
  * **VMess**
  * **Trojan**
  * **Shadowsocks**
  * **SOCKS**

* Мгновенное переключение между сохранёнными профилями.

* Удобная организация большого количества конфигураций.

* Создание отдельных наборов параметров для различных сценариев использования.

## Хранение данных и конфиденциальность

В **Happ** применяется локальный принцип хранения информации: все профили, параметры соединений и пользовательские настройки сохраняются непосредственно на устройстве.

Приложение не отправляет конфигурационные данные внешним сервисам, не передаёт сохранённые параметры сторонним платформам и не осуществляет сбор информации о сетевой активности пользователя. Все настройки остаются под контролем владельца устройства.

## Назначение приложения

**Happ** является клиентским инструментом для управления прокси-конфигурациями и не предоставляет собственные сетевые услуги. Приложение не содержит:

* готовых VPN-сервисов;
* встроенных прокси-узлов;
* собственной серверной инфраструктуры;
* обязательных подписок или автоматических источников доступа.

Для начала работы пользователь самостоятельно добавляет необходимые параметры подключения — например, данные личного сервера или конфигурацию, предоставленную выбранным поставщиком.

Выбор источников, создание профилей и дальнейшее использование настроек выполняются пользователем самостоятельно с учётом требований и законодательства региона.
